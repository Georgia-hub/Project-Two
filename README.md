##Rock Paper Scissors is a classic take 

 The game consits of 3 random options in which the player will either win or loose depeding on what they choose and the computer randomly selects. Each of these options rock, papper or scissors either beats or looses against one another. This is good classic game for easy interactive fun. Game controls

Game rules
